# 💼 Personal Portfolio

A modern and fully responsive personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**. This portfolio showcases my skills, projects, services, and contact information with a clean and professional user interface.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/salmancode-dev/Portfolio.git
```

Navigate to the project

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
